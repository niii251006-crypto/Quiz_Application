# Java Quiz Application

A simple console-based Quiz Application developed in Java using Object-Oriented Programming concepts.

## Features

- Multiple-choice questions
- Calculates final score
- Displays percentage
- Console-based user interface
- Easy to add more questions

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner

## Project Structure

```
Question/
│── Main.java
│── Question.java
│── Quiz.java
│── README.md
```

## How to Run

1. Compile the project:

```
javac *.java
```

2. Run the application:

```
java Main
```
